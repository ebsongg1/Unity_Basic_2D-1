# GameObject

네임스페이스 : `UnityEngine`

## 멤버 변수 / 프로퍼티

| 이름 | 타입 | 설명 |
|---|---|---|
| `transform` | Transform | 이 오브젝트의 Transform 컴포넌트. 모든 GameObject가 반드시 하나 가집니다. |
| `name` | string | 오브젝트의 이름. |
| `tag` | string | 오브젝트에 지정된 태그 문자열. |
| `layer` | int | 오브젝트가 속한 레이어 번호. |
| `activeSelf` | bool | 이 오브젝트 자체의 활성 상태 (읽기 전용). |
| `activeInHierarchy` | bool | 부모까지 고려한 실제 활성 상태 (읽기 전용). |
| `isStatic` | bool | 정적 오브젝트 여부. |
| `scene` | Scene | 이 오브젝트가 속한 씬 (읽기 전용). |

## 멤버 함수 (메서드)

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `SetActive(bool value)` | void | 오브젝트를 활성(true) / 비활성(false)으로 전환합니다. |
| `GetComponent<T>()` | T | 부착된 T 타입 컴포넌트를 가져옵니다. 없으면 null. |
| `GetComponentInChildren<T>()` | T | 자신과 자식들에서 T 컴포넌트를 찾아 반환합니다. |
| `GetComponentInParent<T>()` | T | 자신과 부모들에서 T 컴포넌트를 찾아 반환합니다. |
| `GetComponentsInParent<T>()` | T[] | 자신과 부모들의 모든 T 컴포넌트를 배열로 반환합니다. |
| `GetComponents<T>()` | T[] | 부착된 모든 T 컴포넌트를 배열로 반환합니다. |
| `GetComponentsInChildren<T>()` | T[] | 자신과 자식들의 모든 T 컴포넌트를 배열로 반환합니다. |
| `TryGetComponent<T>(out T component)` | bool | 컴포넌트가 있으면 true와 함께 돌려줍니다. |
| `AddComponent<T>()` | T | T 타입 컴포넌트를 새로 부착합니다. |
| `CompareTag(string tag)` | bool | 지정한 태그와 일치하는지 비교합니다. |

## 정적(static) 멤버

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `GameObject.Find(string name)` | GameObject | 이름으로 오브젝트를 찾습니다. 비용이 크므로 매 프레임 호출은 피합니다. |
| `GameObject.FindWithTag(string tag)` | GameObject | 태그로 오브젝트 하나를 찾습니다. |
| `GameObject.FindGameObjectsWithTag(string tag)` | GameObject[] | 태그로 오브젝트 전부를 찾습니다. |
| `GameObject.CreatePrimitive(PrimitiveType type)` | GameObject | 큐브, 구 등 기본 도형 오브젝트를 생성합니다. |

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Instantiate(Object original, Vector3 position, Quaternion rotation)` | Object | 위치와 회전을 지정하며 복제본을 생성합니다. |
| `Destroy(Object obj, float t)` | void | t초 뒤에 삭제하도록 예약합니다. |
| `DontDestroyOnLoad(Object target)` | void | 씬이 바뀌어도 오브젝트를 파괴하지 않습니다. |
| `FindFirstObjectByType<T>()` | T | 씬에서 T 타입 컴포넌트를 가진 오브젝트를 하나 찾습니다. |
| `FindAnyObjectByType<T>()` | T | 순서를 따지지 않고 더 빠르게 하나 찾습니다. |
| `FindObjectsByType<T>(FindObjectsSortMode sortMode)` | T[] | 씬의 모든 T 타입을 배열로 찾습니다. |

## 상위 클래스(Object)에서 제공되어 함께 자주 쓰는 함수

| 시그니처 | 설명 |
|---|---|
| `Instantiate(Object original)` | 원본의 복제본을 런타임에 생성합니다. (보통 프리팹 복제) |
| `Instantiate(Object original, Transform parent)` | 부모를 지정하며 복제본을 생성합니다. |
| `Destroy(Object obj)` | 오브젝트를 삭제 예약합니다. 즉시 삭제가 아닙니다. |

## 생성자

| 시그니처 | 설명 |
|---|---|
| `new GameObject()` | 빈 GameObject를 생성합니다. |
| `new GameObject(string name)` | 이름을 지정하여 빈 GameObject를 생성합니다. |
