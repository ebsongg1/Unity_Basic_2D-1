# Collider2D

Collider2D는 2D 충돌 영역을 정의하는 컴포넌트들의 공통 부모 클래스입니다.
실제로는 BoxCollider2D, CircleCollider2D, CapsuleCollider2D 등
구체적인 모양의 콜라이더를 사용하며, 이들은 아래 공통 멤버를 모두 가집니다.

네임스페이스 : `UnityEngine`

## Collider2D 공통 멤버 변수 / 프로퍼티

| 이름 | 타입 | 설명 |
|---|---|---|
| `enabled` | bool | 콜라이더 활성화 여부. |
| `isTrigger` | bool | 트리거 여부. true면 물리적으로 막지 않고 통과시키며 이벤트만 발생. |
| `offset` | Vector2 | 오브젝트 중심에서 콜라이더를 얼마나 옮길지. |
| `bounds` | Bounds | 콜라이더를 감싸는 월드 공간 경계 상자 (읽기 전용). |
| `attachedRigidbody` | Rigidbody2D | 연결된 Rigidbody2D (읽기 전용). |
| `sharedMaterial` | PhysicsMaterial2D | 마찰/탄성을 정의하는 물리 머티리얼. |
| `density` | float | 밀도. |
| `friction` | float | 마찰 계수 (읽기 전용, 물리 머티리얼 기반). |
| `bounciness` | float | 탄성 계수 (읽기 전용, 물리 머티리얼 기반). |
| `shapeCount` | int | 콜라이더를 이루는 도형의 개수 (읽기 전용). |
| `usedByComposite` | bool | CompositeCollider2D에 합쳐져 사용될지 여부. |

## Collider2D 공통 멤버 함수 (메서드)

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `IsTouching(Collider2D collider)` | bool | 지정한 콜라이더와 닿아 있는지 검사합니다. |
| `IsTouchingLayers(int layerMask)` | bool | 지정한 레이어와 닿아 있는지 검사합니다. |
| `OverlapPoint(Vector2 point)` | bool | 한 점이 콜라이더 안에 있는지 검사합니다. |
| `Distance(Collider2D collider)` | ColliderDistance2D | 다른 콜라이더와의 최단 거리/겹침 정보를 반환합니다. |
| `ClosestPoint(Vector2 position)` | Vector2 | 지정 위치에서 가장 가까운 콜라이더 표면 점을 반환합니다. |
| `Cast(Vector2 direction, RaycastHit2D[] results, float distance)` | int | 콜라이더를 방향으로 쏴 충돌을 검사합니다. |
| `Raycast(Vector2 direction, RaycastHit2D[] results, float distance)` | int | 콜라이더에서 광선을 쏴 충돌을 검사합니다. |
| `OverlapCollider(ContactFilter2D filter, Collider2D[] results)` | int | 이 콜라이더와 겹친 콜라이더들을 찾습니다. |
| `GetContacts(Collider2D[] colliders)` | int | 현재 닿아 있는 콜라이더들을 가져옵니다. |

## BoxCollider2D

직사각형 모양의 충돌 영역입니다.

| 이름 | 타입 | 설명 |
|---|---|---|
| `size` | Vector2 | 박스의 가로/세로 크기. |
| `edgeRadius` | float | 박스 모서리를 둥글게 만드는 반지름. |

### bounds

`collider.bounds`는 Bounds 구조체이며, 자주 쓰는 멤버는 다음과 같습니다.

| 이름 | 타입 | 설명 |
|---|---|---|
| `center` | Vector3 | 경계 상자의 중심. |
| `size` | Vector3 | 경계 상자의 전체 크기. |
| `extents` | Vector3 | 중심에서 면까지의 거리 (size의 절반). |
| `min` | Vector3 | 경계 상자의 최소 좌표 모서리. |
| `max` | Vector3 | 경계 상자의 최대 좌표 모서리. |

## CircleCollider2D

원 모양의 충돌 영역입니다.

| 이름 | 타입 | 설명 |
|---|---|---|
| `radius` | float | 원의 반지름. |

## CapsuleCollider2D

알약(캡슐) 모양의 충돌 영역입니다. 캐릭터에 자주 사용됩니다.

| 이름 | 타입 | 설명 |
|---|---|---|
| `size` | Vector2 | 캡슐의 가로/세로 크기. |
| `direction` | CapsuleDirection2D | 캡슐이 길어지는 방향. Vertical / Horizontal. |

## 충돌 관련 이벤트 함수 (MonoBehaviour에서 작성)

콜라이더와 관련된 충돌은 스크립트에서 아래 이벤트 함수로 감지합니다.

| 함수 | 호출 시점 |
|---|---|
| `OnCollisionEnter2D(Collision2D)` | 콜라이더끼리 충돌하기 시작할 때. |
| `OnCollisionStay2D(Collision2D)` | 충돌이 유지되는 동안. |
| `OnCollisionExit2D(Collision2D)` | 충돌이 끝날 때. |
| `OnTriggerEnter2D(Collider2D)` | 트리거 영역에 들어올 때. |
| `OnTriggerStay2D(Collider2D)` | 트리거 영역 안에 머무는 동안. |
| `OnTriggerExit2D(Collider2D)` | 트리거 영역에서 나갈 때. |
