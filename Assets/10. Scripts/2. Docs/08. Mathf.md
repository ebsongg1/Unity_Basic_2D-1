# Mathf

Mathf는 게임에서 자주 쓰는 수학 계산 함수들을 모아둔 정적(static) 클래스입니다.
객체를 만들지 않고 `Mathf.` 로 바로 사용합니다.

네임스페이스 : `UnityEngine`

## 정적(static) 변수 - 상수

| 이름 | 설명 |
|---|---|
| `Mathf.PI` | 원주율 (약 3.14159). |
| `Mathf.Infinity` | 양의 무한대. |
| `Mathf.NegativeInfinity` | 음의 무한대. |
| `Mathf.Deg2Rad` | 각도(도)를 라디안으로 바꾸는 변환 계수. |
| `Mathf.Rad2Deg` | 라디안을 각도(도)로 바꾸는 변환 계수. |
| `Mathf.Epsilon` | 0이 아닌 가장 작은 실수 값. |

## 정적(static) 메서드 - 기본

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Mathf.Abs(float f)` | float | 절댓값. |
| `Mathf.Sign(float f)` | float | 부호. 양수면 1, 음수면 -1. |
| `Mathf.Min(float a, float b)` | float | 더 작은 값. |
| `Mathf.Max(float a, float b)` | float | 더 큰 값. |
| `Mathf.Clamp(float value, float min, float max)` | float | value를 min~max 범위로 가둔 값. |
| `Mathf.Clamp01(float value)` | float | value를 0~1 범위로 가둔 값. |
| `Mathf.Pow(float f, float p)` | float | f의 p 제곱. |
| `Mathf.Sqrt(float f)` | float | 제곱근. |

| 시그니처 | 반환 | 설명 |

| `Mathf.CeilToInt(float f)` | int | 올림하여 int로 반환. |

## 정적(static) 메서드 - 보간 / 변화

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Mathf.Lerp(a, b, t)` | float | a에서 b까지 t(0~1) 비율로 선형 보간. |
| `Mathf.LerpUnclamped(a, b, t)` | float | t를 제한하지 않는 선형 보간. |
| `Mathf.InverseLerp(a, b, value)` | float | value가 a~b 사이 어느 비율인지 반환. |
| `Mathf.MoveTowards(current, target, maxDelta)` | float | current를 target 쪽으로 maxDelta만큼 이동. |
| `Mathf.SmoothStep(from, to, t)` | float | 시작/끝이 부드러운 보간. |
| `Mathf.PingPong(t, length)` | float | 0과 length 사이를 왕복하는 값. |
| `Mathf.Repeat(t, length)` | float | 0과 length 사이를 반복하는 값. |
