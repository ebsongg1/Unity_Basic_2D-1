# Color

Color는 색상을 나타내는 구조체(struct)입니다.
빨강(r), 초록(g), 파랑(b), 투명도(a) 네 가지 값으로 구성되며,
각 값은 0.0 ~ 1.0 범위의 실수(float)입니다.

네임스페이스 : `UnityEngine`

## 멤버 변수 / 프로퍼티

| 이름 | 타입 | 설명 |
|---|---|---|
| `r` | float | 빨강 성분 (0.0 ~ 1.0). |
| `g` | float | 초록 성분 (0.0 ~ 1.0). |
| `b` | float | 파랑 성분 (0.0 ~ 1.0). |
| `a` | float | 투명도(알파). 0이면 완전 투명, 1이면 완전 불투명. |
| `grayscale` | float | 색을 회색조로 환산한 밝기 (읽기 전용). |
| `maxColorComponent` | float | r, g, b 중 가장 큰 값 (읽기 전용). |
| `gamma` | Color | 감마 색 공간으로 변환한 색 (읽기 전용). |
| `linear` | Color | 선형 색 공간으로 변환한 색 (읽기 전용). |

## 정적(static) 변수 - 미리 정의된 색상

| 이름 | 색 | 이름 | 색 |
|---|---|---|---|
| `Color.red` | 빨강 | `Color.green` | 초록 |
| `Color.blue` | 파랑 | `Color.white` | 흰색 |
| `Color.black` | 검정 | `Color.yellow` | 노랑 |
| `Color.cyan` | 청록 | `Color.magenta` | 자홍 |
| `Color.gray` | 회색 | `Color.grey` | 회색 (gray와 동일) |
| `Color.clear` | 완전 투명 | | |

## 정적(static) 메서드

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Color.Lerp(a, b, t)` | Color | 색 a에서 b까지 t(0~1) 비율로 선형 보간한 색. |
| `Color.LerpUnclamped(a, b, t)` | Color | t를 0~1로 제한하지 않는 색 보간. |
| `Color.HSVToRGB(h, s, v)` | Color | 색상/채도/명도(HSV)를 RGB Color로 변환. |
| `Color.RGBToHSV(color, out h, out s, out v)` | void | RGB Color를 HSV 값으로 변환. |

## 생성자

| 시그니처 | 설명 |
|---|---|
| `new Color(float r, float g, float b)` | 알파를 1로 두고 색을 생성합니다. |
| `new Color(float r, float g, float b, float a)` | 알파까지 지정하여 색을 생성합니다. |

## 주요 연산자

| 연산 | 의미 |
|---|---|
| `a + b` | 두 색을 성분별로 더함. |
| `a - b` | 두 색을 성분별로 뺌. |
| `c * f` | 색에 실수를 곱해 밝기를 조절. |
| `a * b` | 두 색을 성분별로 곱함. |

## 관련 구조체 - Color32

Color32는 각 성분을 0~255 정수(byte)로 표현하는 색 구조체입니다.
Color와 Color32는 서로 자동 변환됩니다.

| 이름 | 타입 | 설명 |
|---|---|---|
| `r`, `g`, `b`, `a` | byte | 각 색 성분 (0 ~ 255). |
