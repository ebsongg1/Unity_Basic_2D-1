# SpriteRenderer

SpriteRenderer는 2D 이미지인 스프라이트(Sprite)를 씬에 그려주는 컴포넌트입니다.
색상, 좌우/상하 뒤집기, 렌더링 순서 등을 제어할 수 있습니다.

네임스페이스 : `UnityEngine`

## 멤버 변수 / 프로퍼티 - 이미지

| 이름 | 타입 | 설명 |
|---|---|---|
| `sprite` | Sprite | 렌더링할 스프라이트 에셋. |
| `color` | Color | 스프라이트에 입히는 색상과 투명도(틴트). 흰색이면 원본 그대로. |
| `flipX` | bool | 스프라이트를 좌우로 뒤집습니다. |
| `flipY` | bool | 스프라이트를 상하로 뒤집습니다. |

## 멤버 변수 / 프로퍼티 - 렌더링 순서

| 이름 | 타입 | 설명 |
|---|---|---|
| `sortingLayerName` | string | 소속된 Sorting Layer의 이름. |
| `sortingLayerID` | int | 소속된 Sorting Layer의 ID. |
| `sortingOrder` | int | 같은 레이어 안에서의 그리기 순서(Order in Layer). 클수록 앞에 그려집니다. |

## 멤버 변수 / 프로퍼티 - 기타

| 이름 | 타입 | 설명 |
|---|---|---|
| `drawMode` | SpriteDrawMode | 그리기 방식. Simple / Sliced / Tiled. |
| `size` | Vector2 | Sliced / Tiled 모드일 때의 표시 크기. |
| `maskInteraction` | SpriteMaskInteraction | 스프라이트 마스크와의 상호작용 방식. |
| `material` | Material | 렌더링에 사용하는 머티리얼. (Renderer에서 상속) |
| `enabled` | bool | 렌더러 활성화 여부. (Renderer에서 상속) |
| `bounds` | Bounds | 렌더링 영역의 월드 경계 상자 (읽기 전용, Renderer에서 상속). |
| `sharedMaterial` | Material | 여러 오브젝트가 공유하는 머티리얼. (Renderer에서 상속) |
| `isVisible` | bool | 현재 어떤 카메라에든 보이는지 여부 (읽기 전용, Renderer에서 상속). |
| `spriteSortPoint` | SpriteSortPoint | 정렬 기준점. Center / Pivot. |
| `tileMode` | SpriteTileMode | Tiled drawMode일 때 타일 반복 방식. |

## 관련 열거형 (enum)

### SpriteDrawMode
| 값 | 설명 |
|---|---|
| `Simple` | 원본 비율 그대로 그리기. |
| `Sliced` | 9분할로 늘려 그리기. |
| `Tiled` | 타일처럼 반복해 그리기. |

### 관련 열거형 (enum)

#### SpriteMaskInteraction
| 값 | 설명 |
|---|---|
| `None` | 스프라이트 마스크의 영향을 받지 않음. |
| `VisibleInsideMask` | 마스크 안에서만 보임. |
| `VisibleOutsideMask` | 마스크 밖에서만 보임. |

### 관련 이벤트 함수 (MonoBehaviour에서 작성)

| 함수 | 호출 시점 |
|---|---|
| `OnBecameVisible()` | 렌더러가 어떤 카메라에 보이기 시작할 때. |
| `OnBecameInvisible()` | 렌더러가 어떤 카메라에도 보이지 않게 될 때. |
