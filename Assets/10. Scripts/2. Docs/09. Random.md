# Random

Random은 난수(무작위 수)를 만들어 주는 정적(static) 클래스입니다.
객체를 만들지 않고 `Random.` 으로 바로 사용합니다.

주의 : 여기서 말하는 Random은 `UnityEngine.Random` 입니다.
C#의 `System.Random` 과는 다른 클래스이므로, 두 네임스페이스를 함께
사용할 때는 이름 충돌에 유의하세요.

네임스페이스 : `UnityEngine`

## 정적(static) 변수 / 프로퍼티

| 이름 | 타입 | 설명 |
|---|---|---|
| `Random.value` | float | 0.0 ~ 1.0 사이의 무작위 실수 (읽기 전용). |
| `Random.insideUnitCircle` | Vector2 | 반지름 1인 원 안의 무작위 점 (읽기 전용). |
| `Random.insideUnitSphere` | Vector3 | 반지름 1인 구 안의 무작위 점 (읽기 전용). |
| `Random.onUnitSphere` | Vector3 | 반지름 1인 구 표면의 무작위 점 (읽기 전용). |
| `Random.rotation` | Quaternion | 무작위 회전 값 (읽기 전용). |
| `Random.state` | Random.State | 현재 난수 생성기의 상태. 저장/복원에 사용. |
| `Random.rotationUniform` | Quaternion | 더 균일하게 분포된 무작위 회전 (읽기 전용). |

## 정적(static) 메서드

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Random.Range(int min, int max)` | int | min 이상 max 미만의 무작위 정수. max는 포함되지 않습니다. |
| `Random.Range(float min, float max)` | float | min 이상 max 이하의 무작위 실수. max가 포함됩니다. |
| `Random.InitState(int seed)` | void | 시드를 지정해 난수 순서를 고정합니다. |
| `Random.ColorHSV()` | Color | 무작위 색을 생성합니다. |
| `Random.ColorHSV(float hueMin, float hueMax)` | Color | 색상(H) 범위를 지정한 무작위 색. |
| `Random.ColorHSV(float hueMin, float hueMax, float satMin, float satMax, float valMin, float valMax)` | Color | 색상/채도/명도 범위를 모두 지정한 무작위 색. |

## Range 사용 시 주의 - 정수와 실수의 차이

같은 `Range`라도 정수형과 실수형의 최댓값 처리가 다릅니다.

| 호출 | 범위 | max 포함 여부 |
|---|---|---|
| `Random.Range(0, 5)` | 0, 1, 2, 3, 4 | 포함 안 됨 |
| `Random.Range(0f, 5f)` | 0.0 ~ 5.0 | 포함됨 |

배열 인덱스를 무작위로 고를 때는 정수형 Range를 사용하면
`0 ~ (길이 - 1)` 범위가 자연스럽게 나옵니다.

참고 : `Random.state`를 변수에 저장해 두었다가 다시 대입하면
같은 난수 순서를 재현할 수 있습니다. (리플레이, 디버깅 등에 활용)
