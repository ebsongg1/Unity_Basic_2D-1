# Vector (Vector2 / Vector3)

Vector는 공간에서 위치 또는 방향(과 크기)을 나타내는 구조체(struct)입니다.
- Vector2 : x, y 두 개의 값. 주로 2D에서 사용.
- Vector3 : x, y, z 세 개의 값. 3D 및 2D 모두에서 사용.

2D 프로젝트에서도 Transform.position 등은 Vector3를 사용하므로
Vector3를 기본으로 익히되, Vector2도 함께 알아둡니다.

네임스페이스 : `UnityEngine`

## 정적(static) 변수 - 기본 단위

| 이름 | 값 | 설명 |
|---|---|---|
| `Vector3.zero` | (0, 0, 0) | 원점. 크기가 없는 상태. |
| `Vector3.one` | (1, 1, 1) | 모든 축이 1. GameObject의 기본 Scale 값. |

## 정적(static) 변수 - 방향

| 이름 | 값 | 설명 |
|---|---|---|
| `Vector3.right` | (1, 0, 0) | X축 양의 방향 (오른쪽). |
| `Vector3.left` | (-1, 0, 0) | X축 음의 방향 (왼쪽). |
| `Vector3.up` | (0, 1, 0) | Y축 양의 방향 (위쪽). |
| `Vector3.down` | (0, -1, 0) | Y축 음의 방향 (아래쪽). |
| `Vector3.forward` | (0, 0, 1) | Z축 양의 방향 (앞쪽). |
| `Vector3.back` | (0, 0, -1) | Z축 음의 방향 (뒤쪽). |

Vector2도 `zero`, `one`, `right`, `left`, `up`, `down`을 제공합니다.
(Vector2에는 forward / back이 없습니다.)

## 멤버 변수 / 프로퍼티

| 이름 | 타입 | 설명 |
|---|---|---|
| `x` | float | X 좌표. |
| `y` | float | Y 좌표. |
| `z` | float | Z 좌표. (Vector3 전용) |
| `magnitude` | float | 벡터의 길이(크기). 원점에서 얼마나 떨어졌는지 (읽기 전용). |
| `sqrMagnitude` | float | 길이의 제곱. magnitude보다 계산이 빨라 단순 비교용 (읽기 전용). |
| `normalized` | Vector | 길이를 1로 만든 단위 벡터. 방향만 표현 (읽기 전용). |

### 멤버 함수

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Set(float x, float y, float z)` | void | 기존 벡터의 값을 한 번에 바꿉니다. |
| `Normalize()` | void | 벡터 자신을 단위 벡터로 만듭니다. |

## 정적(static) 메서드

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Vector3.Distance(a, b)` | float | 두 점 사이의 거리를 반환합니다. |
| `Vector3.Lerp(a, b, t)` | Vector3 | a에서 b까지 t(0~1) 비율로 선형 보간한 값. |
| `Vector3.LerpUnclamped(a, b, t)` | Vector3 | t를 0~1로 제한하지 않는 선형 보간. |
| `Vector3.MoveTowards(current, target, maxDelta)` | Vector3 | current를 target 쪽으로 maxDelta만큼 이동시킨 값. |
| `Vector3.ClampMagnitude(vector, maxLength)` | Vector3 | 벡터의 길이를 maxLength 이하로 제한합니다. |
| `Vector3.SmoothDamp(current, target, ref velocity, smoothTime)` | Vector3 | 부드럽게 따라가는 이동 (카메라 추적 등에 사용). |
| `Vector3.Scale(a, b)` | Vector3 | 두 벡터를 성분별(x*x, y*y, z*z)로 곱합니다. |
| `Vector3.Reflect(inDirection, normal)` | Vector3 | normal을 기준으로 반사된 방향을 반환합니다. |
| `Vector3.Min(a, b)` | Vector3 | 성분별로 더 작은 값을 모은 벡터. |
| `Vector3.Max(a, b)` | Vector3 | 성분별로 더 큰 값을 모은 벡터. |
| `Vector3.Dot(a, b)` | float | 두 벡터의 내적. 방향 유사도 판단에 사용. |
| `Vector3.Cross(a, b)` | Vector3 | 두 벡터의 외적. (Vector3 전용) |
| `Vector3.Angle(a, b)` | float | 두 벡터 사이의 각도(도). |

Vector2도 `Distance`, `Lerp`, `MoveTowards`, `Dot`, `Angle` 등을 제공합니다.

## 주요 연산자

| 연산 | 의미 |
|---|---|
| `a + b` | 벡터의 합. 두 방향을 합칠 때. |
| `a - b` | 벡터의 차. b에서 a로 향하는 방향. |
| `v * f` | 벡터에 실수 곱. 길이(크기)를 늘이거나 줄일 때. |
| `v / f` | 벡터를 실수로 나눔. |
| `a == b` / `a != b` | 두 벡터가 같은지 비교. |

벡터끼리의 합/차는 가능하지만, 벡터와 단일 실수의 합/차는 불가능합니다.
벡터와 단일 실수의 곱/나눗셈은 가능합니다.
