# Rigidbody2D

Rigidbody2D는 GameObject에 2D 물리(중력, 힘, 충돌 반응)를 적용하는
컴포넌트입니다. 위치 이동은 FixedUpdate에서 처리하는 것이 권장됩니다.

(3D용으로는 Rigidbody가 따로 있습니다. 2D 프로젝트에서는 Rigidbody2D를 사용합니다.)

네임스페이스 : `UnityEngine`

## 멤버 변수 / 프로퍼티 - 운동 상태

| 이름 | 타입 | 설명 |
|---|---|---|
| `linearVelocity` | Vector2 | 현재 이동 속도. (구버전 이름 : velocity) |
| `angularVelocity` | float | 현재 회전 속도(도/초). |
| `position` | Vector2 | 물리 기준 위치. |
| `rotation` | float | 물리 기준 회전 각도(도). |

## 멤버 변수 / 프로퍼티 - 물리 설정

| 이름 | 타입 | 설명 |
|---|---|---|
| `bodyType` | RigidbodyType2D | 몸의 종류. Dynamic / Kinematic / Static. |
| `gravityScale` | float | 중력의 영향 배율. 0이면 중력 없음. |
| `mass` | float | 질량. |
| `linearDamping` | float | 이동 저항(감속). (구버전 이름 : drag) |
| `angularDamping` | float | 회전 저항(감속). (구버전 이름 : angularDrag) |
| `freezeRotation` | bool | 회전을 잠글지 여부. |
| `constraints` | RigidbodyConstraints2D | 위치/회전 축을 고정하는 제약. |
| `simulated` | bool | 물리 시뮬레이션 적용 여부. |
| `interpolation` | RigidbodyInterpolation2D | 프레임 사이 움직임을 부드럽게 보정하는 방식. |
| `collisionDetectionMode` | CollisionDetectionMode2D | 충돌 감지 방식. 빠른 물체는 Continuous 권장. |
| `centerOfMass` | Vector2 | 로컬 기준 무게 중심. |
| `worldCenterOfMass` | Vector2 | 월드 기준 무게 중심 (읽기 전용). |
| `useFullKinematicContacts` | bool | Kinematic 몸도 충돌 이벤트를 받게 할지 여부. |

## 멤버 함수 (메서드)

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `AddForce(Vector2 force, ForceMode2D mode)` | void | 힘을 가합니다. |
| `AddForceX(float force, ForceMode2D mode)` | void | X축으로만 힘을 가합니다. |
| `AddForceY(float force, ForceMode2D mode)` | void | Y축으로만 힘을 가합니다. |
| `AddTorque(float torque)` | void | 회전시키는 힘을 가합니다. |
| `MovePosition(Vector2 position)` | void | 다음 물리 스텝에 지정 위치로 이동시킵니다. |
| `MoveRotation(float angle)` | void | 다음 물리 스텝에 지정 각도로 회전시킵니다. |
| `Cast(Vector2 direction, RaycastHit2D[] results, float distance)` | int | 자신의 콜라이더를 방향으로 쏴 충돌을 검사합니다. |
| `IsTouching(Collider2D collider)` | bool | 지정 콜라이더와 닿아 있는지 검사합니다. |
| `Sleep()` | void | 물리 계산을 잠시 멈춥니다. |
| `WakeUp()` | void | 멈춘 물리 계산을 다시 깨웁니다. |
| `AddForceAtPosition(Vector2 force, Vector2 position)` | void | 특정 위치에 힘을 가합니다(회전이 함께 발생). |
| `AddRelativeForce(Vector2 relativeForce)` | void | 오브젝트 자신의 방향 기준으로 힘을 가합니다. |
| `GetPointVelocity(Vector2 point)` | Vector2 | 특정 점의 현재 속도를 반환합니다. |
| `IsSleeping()` | bool | 물리 계산이 멈춰 있는지 검사합니다. |
| `IsAwake()` | bool | 물리 계산이 동작 중인지 검사합니다. |
| `OverlapCollider(ContactFilter2D filter, Collider2D[] results)` | int | 겹친 콜라이더들을 찾습니다. |

## 관련 열거형 (enum)

### RigidbodyType2D
| 값 | 설명 |
|---|---|
| `Dynamic` | 중력과 힘, 충돌에 모두 반응하는 일반 물리 몸. |
| `Kinematic` | 물리 엔진이 건드리지 않음. 코드로만 움직임. |
| `Static` | 움직이지 않는 고정 몸. |

### ForceMode2D
| 값 | 설명 |
|---|---|
| `Force` | 질량을 고려한 지속적인 힘. |
| `Impulse` | 질량을 고려한 순간적인 힘(충격). |

### 추가 열거형 (enum)

#### RigidbodyConstraints2D
| 값 | 설명 |
|---|---|
| `None` | 제약 없음. |
| `FreezePositionX` | X 위치 고정. |
| `FreezePositionY` | Y 위치 고정. |
| `FreezeRotation` | 회전 고정. |
| `FreezeAll` | 위치와 회전 모두 고정. |

#### RigidbodyInterpolation2D
| 값 | 설명 |
|---|---|
| `None` | 보간 없음. |
| `Interpolate` | 이전 프레임을 기준으로 부드럽게 보정. |
| `Extrapolate` | 다음 위치를 예측하여 보정. |

#### CollisionDetectionMode2D
| 값 | 설명 |
|---|---|
| `Discrete` | 일반적인 충돌 감지 (기본값). |
| `Continuous` | 빠르게 움직이는 물체의 통과(터널링)를 막는 정밀 감지. |
