## 마크다운 언어란?
마크업(Mark-Up) 언어의 경량화된 버전으로
현대 문서 편집기에 일반적으로 사용되는
언어이다. 마크다운 언어로 텍스트를 
작성하면 해당 소스를 프로그램에서 읽어서
변환 후 사용자에 따라서 맞춤형 출력이
가능하다.
보통 개발자들이 주로 사용하며,
게임회사의 경우 프로젝트의 위키를
제작하는데에 사용된다. 프로그래머, 기획자,
디자이너들이 주로 사용한다

## 문법

### 1. 제목(Heading)
- `#`의 개수로 수준(Heading Level)을 구분한다.
```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
```
- 일반적으로는 2단계 ~ 3단계 까지 사용

# 제목 1
## 제목 2
### 제목 3
#### 제목 4

### 문단과 줄 바꿈
- 보통 한줄을 띄우면 새로운 문단이 됩니다.
- 강제 줄바꿈은 문장 끝에 `두개의 공백 + Enter`를 입력합니다

```
문단 1입니다.
문단 2입니다.
```

문단 1입니다.  
문단 2입니다.

### 4. 목록(List)
#### 순서없는 목록
```
- 항목 1
	- 하위 항목 1 (Tab, Shift + Tab)
	- 하위 항목 2
* 항목 2
+ 항목 3
```
- 항목 1
	- 하위 항목 1 (Tab, Shift + Tab)
	- 하위 항목 2
* 항목 2
+ 항목 3

#### 순서 있는 목록
```
1. 첫번째
2. 두번째
	1. 하위 첫번째 (Tab, Shift + Tab)
	2. 하위 두번째
```
1. 첫번째
2. 두번째
	1. 하위 첫번째 (Tab, Shift + Tab)
	2. 하위 두번째
