# Transform

Transform는 GameObject의 위치(Position), 회전(Rotation), 크기(Scale)를
담당하는 컴포넌트입니다. 모든 GameObject가 반드시 하나씩 가집니다.
부모-자식 계층 구조를 만들 수 있으며, 자식은 부모의 변환에 영향을 받습니다.

네임스페이스 : `UnityEngine`

## 멤버 변수 / 프로퍼티 - 위치, 회전, 크기

| 이름 | 타입 | 설명 |
|---|---|---|
| `position` | Vector3 | 월드 공간 기준 위치. |
| `localPosition` | Vector3 | 부모 기준의 상대 위치. |
| `rotation` | Quaternion | 월드 공간 기준 회전. |
| `localRotation` | Quaternion | 부모 기준의 상대 회전. |
| `eulerAngles` | Vector3 | 월드 회전을 각도(도) 형태로 표현한 값. |
| `localEulerAngles` | Vector3 | 부모 기준 회전을 각도 형태로 표현한 값. |
| `localScale` | Vector3 | 부모 기준의 크기. 인스펙터의 Scale 값. |
| `lossyScale` | Vector3 | 부모까지 반영된 월드 크기 (읽기 전용). |

## 멤버 변수 / 프로퍼티 - 방향 벡터 (읽기 전용)

| 이름 | 타입 | 설명 |
|---|---|---|
| `right` | Vector3 | 오브젝트의 오른쪽 방향 (월드 기준). |
| `up` | Vector3 | 오브젝트의 위쪽 방향 (월드 기준). |
| `forward` | Vector3 | 오브젝트의 앞쪽 방향 (월드 기준). |

## 멤버 변수 / 프로퍼티 - 계층 구조

| 이름 | 타입 | 설명 |
|---|---|---|
| `parent` | Transform | 부모 Transform. 없으면 null. |
| `root` | Transform | 계층의 최상위 Transform. |
| `childCount` | int | 직속 자식의 개수 (읽기 전용). |

## 멤버 함수 (메서드)

| 시그니처 | 반환 | 설명 |
|---|---|---|
| `Translate(Vector3 translation)` | void | 전달된 벡터만큼 위치를 이동시킵니다. |
| `Rotate(Vector3 eulers)` | void | 전달된 각도만큼 회전시킵니다. |
| `RotateAround(Vector3 point, Vector3 axis, float angle)` | void | 특정 점을 중심으로 회전시킵니다. |
| `SetPositionAndRotation(Vector3 position, Quaternion rotation)` | void | 위치와 회전을 한 번에 설정합니다. |
| `LookAt(Transform target)` | void | 대상을 바라보도록 회전시킵니다. |
| `SetParent(Transform parent)` | void | 부모를 설정합니다. |
| `GetChild(int index)` | Transform | index번째 자식 Transform을 반환합니다. |
| `IsChildOf(Transform parent)` | bool | 지정한 Transform의 자식인지 검사합니다. |
| `Find(string name)` | Transform | 이름이 일치하는 자식 Transform을 찾습니다. |
| `DetachChildren()` | void | 모든 자식을 부모-자식 관계에서 분리합니다. |
| `GetSiblingIndex()` | int | 형제들 사이에서 자신의 순서를 반환합니다. |
| `SetSiblingIndex(int index)` | void | 형제들 사이에서 자신의 순서를 바꿉니다. |
| `SetAsFirstSibling()` | void | 형제들 중 첫 번째 순서로 보냅니다. |
| `SetAsLastSibling()` | void | 형제들 중 마지막 순서로 보냅니다. |
| `TransformPoint(Vector3 position)` | Vector3 | 로컬 좌표를 월드 좌표로 변환합니다. |
| `InverseTransformPoint(Vector3 position)` | Vector3 | 월드 좌표를 로컬 좌표로 변환합니다. |
| `TransformDirection(Vector3 direction)` | Vector3 | 로컬 방향을 월드 방향으로 변환합니다. |
